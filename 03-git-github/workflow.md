**# Git Workflow**



**My standard Git workflow follows these steps:**



**1. Edit files**

**2. Check status using git status**

**3. Stage changes using git add**

**4. Commit changes with a clear message**

**5. Push changes to GitHub**



**## Commands Used**

**- git status**

**- git add**

**- git commit**

**- git push**



**## Viewing History and Changes**

**- git log shows commit history**

**- git diff shows file changes**

**- git show displays details of a specific commit**



**Following this workflow keeps my commits clean and organized.


## Branching Practice**

**I created a feature branch, made changes, committed them,**

**and merged the branch back into main.**
