**# Handling Merge Conflicts**



**Merge conflicts occur when Git cannot automatically merge changes.**



**## How I Handle Conflicts**

**1. Identify conflicted files.**

**2. Open the file and look for conflict markers.**

**3. Decide which changes to keep.**

**4. Remove conflict markers.**

**5. Stage the resolved file.**

**6. Commit the resolution.**



**## Conflict Markers**

**- <<<<<<< HEAD**

**- =======**

**- >>>>>>> branch-name**



**Conflicts are resolved manually and carefully to avoid breaking code.**
